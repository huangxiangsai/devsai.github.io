关闭当前窗口
Ctrl+w c 或者 :q

在只有一个窗口时， Ctrl+w c将不起作用 ，只能用:q或者Ctrl+w q来关闭窗口

如果我们需要看同一文件的两个部分的内容， 文件又很长， 这时我们需要再打开一个相同的文件 ，我们可以使用Ctrl+w s 或 Ctrl+w v(一个是上下分屏，另一个是左右分屏)命令实现

当我们已经打开一个文件后，才想到需要查看另一文件 ，我们就可以使用如下命令：

垂直分屏: :sp filename

水平分屏：:vsp filename
